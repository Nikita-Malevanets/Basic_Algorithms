# Висновки до Завдання 2 (Метод Монте-Карло)

## Результати
Було проведено обчислення визначеного інтеграла функції f(x) = x^2 на відрізку [0, 2].

1. **Точне значення (функція `quad`):** 2.666666666666667
2. **Метод Монте-Карло (100,000 точок):** ~2.66 (результат змінюється в межах похибки при кожному запуску)

## Висновок
Метод Монте-Карло показав результат, дуже близький до точного аналітичного розрахунку. Різниця між значеннями (похибка)
є незначною.

**Спостереження:**

- Метод Монте-Карло є ймовірнісним, його точність залежить від кількості випадкових точок (N).
- Чим більша кількість точок, тим менша похибка.
- Цей метод ефективний, коли аналітичне обчислення є складним або неможливим, хоча для простих функцій він працює
  повільніше за аналітичні алгоритми.
